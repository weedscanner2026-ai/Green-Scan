# Reports & Analytics Guide

## Overview
The Reports & Analytics page provides visual insights into your user base with interactive charts and graphs.

---

## Features

### 📊 Visual Charts

1. **User Distribution by Type (Pie Chart)**
   - Shows breakdown of Students, Agriculturists, and Others
   - Interactive hover to see exact numbers
   - Color-coded for easy identification

2. **Account Status (Pie Chart)**
   - Active vs Inactive users
   - Percentage breakdown
   - Quick overview of user engagement

3. **Registrations Over Time (Line Chart)**
   - Last 30 days of registration activity
   - Trend visualization
   - Identify peak registration periods

4. **Monthly Registrations (Bar Chart)**
   - Last 12 months overview
   - Compare month-to-month growth
   - Identify seasonal patterns

### 📈 Statistics Cards

- **Total Users** - All registered users
- **Active Users** - Currently active accounts with percentage
- **Active This Week** - Users who logged in last 7 days
- **With Institution** - Users who provided institution info

### 🏫 Top Institutions

- Bar chart showing top 10 institutions
- Number of users per institution
- Visual comparison of institution popularity

### 📋 Summary Statistics

- Detailed breakdown by user type
- Percentage calculations
- Inactive user count
