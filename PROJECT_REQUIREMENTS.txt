================================================================================
GREEN SCAN - WEED DETECTION SYSTEM
PROJECT REQUIREMENTS DOCUMENT
================================================================================

PROJECT OVERVIEW:
An AI-powered weed detection system for Philippine agriculture consisting of:
- Android mobile application for real-time weed identification
- Web-based admin panel for model training and user management
- Machine learning model using TensorFlow/Keras
- User registration and authentication system

================================================================================
SYSTEM REQUIREMENTS
================================================================================

1. HARDWARE REQUIREMENTS
------------------------
Development Machine:
- Processor: Intel Core i5 or higher (or equivalent AMD)
- RAM: 8GB minimum, 16GB recommended
- Storage: 10GB free space minimum
- GPU: NVIDIA GPU with CUDA support (optional, for faster training)

Android Device:
- Android 7.0 (API Level 24) or higher
- Camera with autofocus
- 2GB RAM minimum
- 100MB free storage

2. SOFTWARE REQUIREMENTS
------------------------

A. Python Environment:
   - Python 3.8 - 3.10 (tested on 3.10.9)
   - pip package manager

B. Android Development:
   - Android Studio Arctic Fox or later
   - JDK 11 or higher
   - Android SDK (API Level 24+)
   - Gradle 7.0+

C. Operating System:
   - Windows 10/11 (tested)
   - macOS 10.14+ (compatible)
   - Linux Ubuntu 18.04+ (compatible)

================================================================================
DATASET REQUIREMENTS
================================================================================

Training Data:
- Minimum 100 images per weed class
- Recommended 500-1000 images per class
- Image format: JPG, PNG
- Resolution: 224x224 pixels (auto-resized)
- Balanced dataset across all classes

Weed Classes (Current):
1. Carabao Grass (Paspalum conjugatum)
2. Cogon Grass (Imperata cylindrica)
3. Makahiya (Mimosa pudica)
4. Morsikos (Bidens pilosa)
5. Plagtiki/Teki (Cyperus rotundus)
6. Not Weed (negative samples)

================================================================================
INSTALLATION STEPS
================================================================================

1. Install Python 3.10.9
2. Install required Python packages: pip install -r requirements.txt
3. Install Android Studio
4. Configure JDK and Android SDK
5. Clone/download project files
6. Prepare dataset in dataset/weeds/ folder
7. Train model: python train_model.py
8. Start admin server: python unified_admin_server.py
9. Open Android project in Android Studio
10. Build and install APK on device
